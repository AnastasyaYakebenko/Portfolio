Мои проекты
1. Мобильное приложение Stets Home для умного дома
Суть: Комплексный проект по разработке приложения для управления инновационными умными устройствами (с функцией энергосбережения) и интеграции с другими гаджетами. Артефакты: Use Case, DFD, ER-диаграммы, словарь данных, карта пользовательских историй, интерактивный прототип в Figma, ПМИ, видео-защита. Репозиторий с деталями: Мобильное приложение "Умный дом"

2. Моделирование процессов сети клиник
Суть: Автоматизация записи пациентов в медклинике. Анализ AS-IS и проектирование TO-BE процессов. Артефакты: BPMN диаграммы, UML State Machine, User Story Mapping, Data Flow Diagram, интерактивный прототип, сценарии Use Case в Figma. Репозиторий с деталями: Проект "Клиника"

3. Проектирование интернет-магазина
Суть: Разработка требований и моделирование данных для сайта интернет-магазина. Артефакты: Use Case диаграммы, логические ER-диаграммы в нотациях Чена и Crow's Foot, нормализация данных до 3НФ. Репозиторий с деталями: Проект "Интернет-магазин"

4. Новый функционал для приложения изучения языков
Суть: Расширение приложения модулем "Карточки для запоминания слов" с видео-контекстом. Артефакты: Спецификация требований (SRS), Use Case диаграммы, функциональные и нефункциональные требования, бизнес-правила, глоссарий. Репозиторий с деталями: Проект "Языковое приложение"

5. Интеграция ТВ-каналов в онлайн-кинотеатр
Суть: Добавление нового раздела "ТВ-каналы" в существующий онлайн-кинотеатр. Артефакты: Анализ UML Component Diagram, разработка UML Sequence Diagram для нового сервиса. Репозиторий с деталями: Проект "Онлайн-кинотеатр"

6. Тестирование сайта бронирования отелей
Суть: Комплексное тестирование и подготовка приемочной документации для сайта бронирования. Артефакты: План тестирования и тест-кейсы, Программа и методика испытаний (ПМИ), Руководство пользователя. Репозиторий с деталями: Проект "Тестирование"

7. Моделирование финансового процесса в IDEF0 (Стоматология)
Суть: Анализ и оптимизация процесса движения денежных средств в стоматологической клинике на этапе до выплаты заработной платы с использованием функциональной модели. Артефакты: IDEF0-диаграмма, описание целей и задач моделирования. Результаты: Выявлены узкие места и дублирование функций, предложена оптимизация, сокращающая время прохождения документов с 5 до 2 дней. Репозиторий с деталями: Проект "IDEF0-моделирование"
