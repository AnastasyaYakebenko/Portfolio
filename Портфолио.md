## Мои проекты

### 1. Мобильное приложение Stets Home для умного дома
**Суть:** Комплексный проект по разработке приложения для управления инновационными умными устройствами (с функцией энергосбережения) и интеграции с другими гаджетами.
**Артефакты:** Use Case, DFD, ER-диаграммы, словарь данных, карта пользовательских историй, интерактивный прототип в Figma, ПМИ, видео-защита.
**Репозиторий с деталями:** [Проект Stets Home](https://github.com/AnastasyaYakebenko/-/tree/main) (ссылка ведет на ваш репозиторий с файлами проекта)

### 2. Моделирование процессов сети клиник
**Суть:** Автоматизация записи пациентов в медклинике. Анализ AS-IS и проектирование TO-BE процессов.
**Артефакты:** BPMN диаграммы, UML State Machine, User Story Mapping, Data Flow Diagram, интерактивный прототип, сценарии Use Case в Figma.
**Репозиторий с деталями:** [Проект "Клиника"](https://github.com/AnastasyaYakebenko/-/tree/main) (ссылка ведет на тот же репозиторий, можно уточнить названием папки)

### 3. Проектирование интернет-магазина
**Суть:** Разработка требований и моделирование данных для сайта интернет-магазина.
**Артефакты:** Use Case диаграммы, логические ER-диаграммы в нотациях Чена и Crow's Foot, нормализация данных до 3НФ.
**Репозиторий с деталями:** [Проект "Интернет-магазин"](https://github.com/AnastasyaYakebenko/-/tree/main) (ссылка ведет на ваш репозиторий с файлами)

### 4. Новый функционал для приложения изучения языков

**Артефакты:** Спецификация требований (SRS), Use Case диаграммы, функциональные и нефункциональные требования, бизнес-правила, глоссарий.
**Репозиторий с деталями:** [Проект "Языковое приложение"](https://github.com/AnastasyaYakebenko/-/tree/main) (ссылка ведет на ваш репозиторий)
